## Troubleshooting

### Can't Sign In

**Problem**: Google Sign-In not working

**Solutions:**
1. Check your internet connection
2. Try a different browser
3. Clear browser cache
4. Disable ad blockers temporarily
5. Check if pop-ups are blocked

### No Search Results

**Problem**: Search returns empty

**Solutions:**
1. Try a broader search term
2. Check spelling
3. Remove special characters
4. Try a different neighborhood

### Payment Issues

**Problem**: Payment failed or credits not added

**Solutions:**
1. Check your MercadoPago account
2. Wait a few minutes (can take up to 5 minutes)
3. Refresh the page
4. Contact support if issue persists

### Slow Performance

**Problem**: Website is slow

**Solutions:**
1. Check your internet speed
2. Close other browser tabs
3. Clear browser cache
4. Try incognito mode
